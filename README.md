# Payroll Processing System

Consolidated payroll automation system for batch payment processing.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deployment

Push to GitHub and connect to Vercel.

Add environment variables in Vercel:
- DATABASE_URL
- JWT_SECRET
- NEXTAUTH_SECRET
